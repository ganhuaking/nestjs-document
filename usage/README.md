# How to use NestJs

## nestjs-cli tool installation

for npm user:

```shell==
npm i -g @nestjs/cli
```

for yarn user:
```shell==
yarn global add @nestjs/cli
```

## nestjs cli new project

```shell==
nest new $Project_Name
```

## nestjs cli command lookup

```shell==
nest --help
```
