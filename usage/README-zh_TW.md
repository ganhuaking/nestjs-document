# 如何使用 NestJs

[English Version](usage/README.md "English Version")

## 安裝 nestjs-cli 

使用 npm 安裝:

```shell==
npm i -g @nestjs/cli
```

使用 yarn 安裝:
```shell==
yarn global add @nestjs/cli
```

## 使用 nestjs cli 建立新的專案 

```shell==
nest new $Project_Name
```

## 查訊 nestjs 可用指令

```shell==
nest --help
```
