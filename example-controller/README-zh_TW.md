## TasksController

### 使用 nestjs-cli 產生 TasksController

```shell
nest g controller tasks --no-spec
```
**Notice** 使用 --no-spec 代表不要產生 .spec.ts 測試檔案 

![](https://i.imgur.com/gsWw0CF.png)

### 使用 nestjs-cli 產生 AuthController

```shell
nest g controller auth --no-spec
```

**Notice** 使用 --no-spec 代表不要產生 .spec.ts 測試檔案
